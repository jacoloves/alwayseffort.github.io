## 今日やったこと
Rustでアルゴリズムを解いた。   
大きな数なのでかけた値に10^9+7で割ってあげれば良い。   
$
x = x * i \bmod 10^9 + 7 
$

```rs
fn abc055_b() {
    input! {
        n: i64,
    }

    let amod = 1000000007;

    let mut ans = 1;

    for i in 1..=n {
        ans *= i;
        ans %= amod;
    }

    println!("{}", ans);
}
```
